## Add Welcome Video to Company Invite/Join Page

Show an optional video placeholder on the `/join` page — left of the signup form on desktop, above it on mobile. Admins set the URL per company.

### Database

New migration adds a column and updates the lookup RPC:

- `ALTER TABLE public.companies ADD COLUMN join_video_url text;`
- `DROP` + `CREATE OR REPLACE FUNCTION public.lookup_company_by_invite_token(_token uuid)` to also return `join_video_url`.

### Shared Video Embed Component

Extract the existing `VideoEmbed` from `src/pages/CoursePage.tsx` into `src/components/VideoEmbed.tsx`. It already supports YouTube, Vimeo, Loom, Descript, and raw video files — the same parser used elsewhere in the app. `CoursePage` is refactored to import from the new location.

### Join Page (`src/pages/Join.tsx`)

- Extend `PendingCompany` with `join_video_url: string | null` and read it from the RPC response.
- Replace the centered `max-w-md` wrapper with a two-column responsive layout:
  - **Desktop (`md:`)**: video on the left, signup form on the right, both centered vertically. Max width ~6xl.
  - **Mobile**: video stacks above the form.
- When `join_video_url` is empty, the form remains centered as it is today (no empty placeholder shown).
- When present, render `<VideoEmbed url={...} />` inside a rounded container, with a small caption like "A quick welcome from {company}".

### Admin Panel

Add a **"Welcome Video URL"** field to the Edit Company dialog in `src/components/admin/CompaniesTab.tsx`:

- New text input with placeholder `https://youtu.be/... or Loom/Vimeo/Descript link`
- Helper text: "Shown to invited users on the join page. Supports YouTube, Vimeo, Loom, Descript, or direct MP4."
- Wire into the existing `updateMutation` so it writes `join_video_url` (or null when empty).
- Pre-populate `editJoinVideoUrl` in `openEdit`.
- Update the SELECT in the list query to include `join_video_url`.

### Where to add the link (admin)

Navigate to **Admin Panel → Companies tab → click the pencil/edit icon on a company row → "Welcome Video URL" field → Save**.

### Technical Details

- Files modified: `src/pages/Join.tsx`, `src/components/admin/CompaniesTab.tsx`, `src/pages/CoursePage.tsx` (import path), `src/integrations/supabase/types.ts` (regenerated by migration).
- Files created: `src/components/VideoEmbed.tsx`, one new SQL migration.
- No RLS changes needed — the RPC is `SECURITY DEFINER` and the column is exposed only through it.
- Layout uses a `grid md:grid-cols-2 gap-8 items-center` container so the form keeps current sizing on the right and the video fills the left column with `aspect-video`.
